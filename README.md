# smart_irrigate_project

Інтелектуальна система автоматичного поливу для теплиць.

Проект Smart Irrigate розробляється для оптимізації споживання води (ціль: зниження до 30%) та підвищення ефективності тепличного господарства шляхом точного контролю вологості ґрунту.

## Прогрес та Ключові Вимоги (MVP - v1.0)

Наш MVP сфокусований на двох основних компонентах:

1.  Апаратна частина (HW): Збір даних з датчиків вологості та керування реле/клапанами.
2.  Програмна частина (SW): Веб-інтерфейс моніторингу, логування даних у PostgreSQL та логіка автополиву.
